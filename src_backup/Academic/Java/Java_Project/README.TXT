PROJECT TITLE: STUDENT GRADE CALCULATOR.
PURPOSE OF PROJECT: TO IMPLEMENT A INTERFACE FOR STUDENT AND FACULTY FOR A GRADES SYSTEM.
VERSION/DATE: 01-05-2025.
HOW TO START THIS PROJECT: COMPILE ALL THE .java FILES AND RUN GradeCalculator.java.
AUTHORS: BHAVANI SHANKAR ARUGONDA.
USER INSTRUCTIONS:FOR AUTHENTICATION OF FACULTY THE PASSWORD IS SAME AS THE SUBJECT NAME.
CLASS DESCRIPTION:
1.Student
     * This class implements the functions related to student.
     * It calculates and prints a particular student grades.

2.Teacher
     * This class implements all the functionalities of a teacher of a particular subject.
     * It stores all the data corresponding to the students into text files corresponding to the subject.

3.GradeCalculator
     * This implements the main function.
     * It validates teachers and brings together all the supporting classes.
