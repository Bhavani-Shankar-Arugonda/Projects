# ToDo Frontend (Vanilla JS)

## Overview
A lightweight, dependency-free frontend interface for managing personal tasks. Designed to integrate seamlessy with the **User Management API**.

## Structure
- `index.html`: Semantic HTML5 structure.
- `index.js`: (Planned) Client-side logic for task management.

## Integration
This project is designed to be served by the [User Management API](../user-management-api) but can be opened directly in a browser for UI testing.
