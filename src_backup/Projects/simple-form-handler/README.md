# Simple Form Handler

## Overview
A Node.js/Express application that demonstrates form processing. It accepts user input via a web interface and saves the submitted data to a local JSON file.

## Features
- **Form Handling**: Parses URL-encoded form data.
- **Data Persistence**: Saves submissions to `data.json`.
- **Validation**: Basic sanitization of input fields.

## Setup
1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Start Server**:
   ```bash
   node form.js
   ```
3. **Usage**:
   Visit [http://localhost:5000](http://localhost:5000) to submit the form.
